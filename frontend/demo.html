💬 Intelligent Querying

Ask questions in natural language and get AI-powered answers with citations.

🖼️

Query Interface

Mode selector, query input, and rich response display

Query Modes:

🔵 Vector Mode - Semantic Search

Best for factual questions about single concepts

Example: "What is NEFT?"

🟣 Graph Mode - Relationship Queries

Best for questions about dependencies and connections

Example: "What systems does RTGS depend on?"

🟢 Hybrid Mode - Combined Approach

Best for complex questions requiring both semantic and structural understanding

Example: "Compare NEFT and RTGS transaction limits and their dependencies"

⚪ Auto Mode - Let AI Decide

System automatically chooses the best mode for your question

Example: Any question - the AI will route it appropriately

📝 Rich Response Display

🖼️

Response Area

Answer with citations, metrics, and source references

What you get:

  • AI-Generated Answer: Clear, concise response to your question
  • Source Citations: References to specific documents and sections
  • Faithfulness Score: Confidence metric (0-100%) showing answer accuracy
  • Query Metrics: Execution time and chunks retrieved
  • Mode Badge: Shows which retrieval mode was used
  • Warnings: Alerts for low confidence or insufficient data

🔧 Technical Architecture

Frontend Stack:

  • Pure HTML, CSS, and JavaScript (no frameworks)
  • Responsive design for all devices
  • Real-time status polling
  • Modern UI with smooth animations

Backend Integration:

  • RESTful API communication
  • File upload with FormData
  • JSON request/response handling
  • CORS-enabled for cross-origin requests

Powered By:

  • Ollama: LLM for answer generation
  • Qdrant: Vector database for semantic search
  • Neo4j: Knowledge graph for relationships
  • PostgreSQL: Document storage
